vision-landing-page

Адаптивный современный лендинг для маркетингового digital-агентства **Vision**. 

# 🚀 Ссылка на проект (Live Demo)
👉 https://izaar-code.github.io/vision-landing-page/

# 🛠 Используемые технологии
 HTML5 — семантическая разметка сайта
 CSS3 — стилизация, Flexbox, Grid Layout
 CSS Animations — плавные микро-взаимодействия и эффект пульсации для кнопок
 БЭМ (Блок-Элемент-Модификатор) — методология именования классов

# 📱 Особенности проекта
 Полная адаптивность (Pixel Perfect под десктопы, планшеты и мобильные экраны).
 Оптимизированный маркетинговый контент на русском языке.
 Слайдер в секции отзывов (Testimonials).
 Интерактивные элементы меню и кликабельные CTA-кнопки.
